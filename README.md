# BC/AD or BCE/CE Era Notation Selector (Chrome extension)

Customize era notations so that they are always BC/AD or always BCE/CE

Customize era notations on any website. 

How to use it:
1. Add this extension to the top bar. 
2. Make sure that the "Use on this website" checkbox is selected for the website where you want to use this functionality. You have to activate this extension for each website individually.
3. Select BC/AD or BCE/CE. You only need to make this choice once. It will be applied globally to all websites that you selected on the previous step. You can change your choice later if you need to.

To install visit this page: https://chromewebstore.google.com/detail/bcad-or-bcece-era-notatio/fedcnfmkgihadcdfnjfhcgmlmcgnlipn
